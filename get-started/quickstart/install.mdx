## Setup your environment

Follow the steps below to install and use GPT-4 Free on your computer.

### Requirements

You will need to have Python installed. If you don't have it installed, you can download it [here.](https://www.python.org/downloads/)

<Note>We recommend using the latest python version `3.12.0`, or python `3.10+` at least.</Note>
### Installing the `g4f`library

#### Installing via [`pip`](https://pypi.org/project/pip/)

You can install the GPT-4 Free library from PyPi using pip, or pipx to only use the CLI:

<CodeGroup>

```shell pip
pip install -U g4f
```

```shell pip3
pip3 install -U g4f
```

```bash pipx
pipx install g4f
```

</CodeGroup>

#### Installing via [`git-clone`](https://github.com/git-guides/git-clone)

<AccordionGroup>
<Accordion icon="github" title="Clone the repository locally">
If you want to work with the latest version of the GPT-4 Free library or contribute to the project, you can clone the repository. Details on how to clone a repository can be found [here.](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
```bash
git clone https://github.com/xtekky/gpt4free.git
```
</Accordion>
</AccordionGroup>

#### Installing via Docker
### Docker Installation

<AccordionGroup>
<Accordion icon="docker" title="Install Docker and Docker Compose">
If you have Docker installed, you can run the project in a Docker container. First, ensure that you have both Docker and Docker Compose installed. You can install Docker [here](https://docs.docker.com/get-docker/) and Docker Compose [here](https://docs.docker.com/compose/install/).
</Accordion>

<Accordion icon="github" title="Clone the repository">
Clone the Github repository.
```bash
git clone https://github.com/xtekky/gpt4free.git
```
</Accordion>

<Accordion icon="docker" title="Build the Docker image">
Change your directory to the root of the project and build the Docker image.
```bash
cd gpt4free
```
```bash
docker compose build
```
</Accordion>

<Accordion icon="docker" title="Start the service">
Start the service using Docker Compose.
```bash
docker compose up
```
</Accordion>
</AccordionGroup>

## Usage and API Configuration

<AccordionGroup>
<Accordion icon="coding" title="Navigate to the project directory">
After you cloned the repository, navigate to the root directory of the project:
```bash
cd gpt4free
```
</Accordion>

<Accordion icon="terminal" title="Create a Python virtual environment">
It's recommended to create a Python virtual environment for your project. More information can be found in the [Python official documentation](https://docs.python.org/3/tutorial/venv.html).
```bash
python3 -m venv venv
```
</Accordion>

<Accordion icon="coding" title="Activate the virtual environment">
On Windows:
```cmd
.\venv\Scripts\activate
```
On macOS and Linux:
```bash
source venv/bin/activate
```
</Accordion>

<Accordion icon="python" title="Install the required Python packages">
You can install all the necessary packages from `requirements.txt` using this command:
```bash
pip install -r requirements.txt
```
</Accordion>

<Accordion icon="play" title="Run the script">
Create a `test.py` file in the root folder and import the g4f package to start using the repo.
```python
import g4f

...
```
</Accordion>
</AccordionGroup>
