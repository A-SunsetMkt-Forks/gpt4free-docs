## What is g4f ?

G4F, short for gpt4free is a powerful LLM-Cli, on a mission to decentralize and providide free access to AI.

<Warning>"gpt4free" serves as a PoC (proof of concept), demonstrating the development of an API package with multi-provider requests, with features like timeouts, load balance and flow control.</Warning>
